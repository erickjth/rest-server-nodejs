# rest-server-nodejs

NodeJS starter kit based on hexagonal architecture with a REST layer.

# Setup
- Run `yarn install`
- Copy local config `cp src/config/default.local.js src/config/local.js`.
- Edit local.js based on base.js keys.

# Run
- `yarn start:dev`
- `yarn start:prod`
